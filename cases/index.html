動くものから、書いて残す。

Phasera が中小企業向けに作っている AI エージェントの実装と、現場での動作確認をケーススタディとして書き残しています。System prompt と動作サンプルを公開しているので、自分でも作れる形になっています。

  1. 問い合わせメールの一次返信を、AI に下書きさせる現実的なライン士業や中小企業の問い合わせ対応を、AI エージェントで「一次返信の下書きまで」自動化した事例。設計、System Prompt、動作確認、コスト試算まで Phasera が実装ベースで解説します。士業 / 中小企業バックオフィス担当者 · 3154字
  2. 委任状の下書きを2分で整える、士業のための書類エージェント委任状や同意書の下書きを AI エージェントで整え、最終確認だけ士業に残す設計。Edge Function と AI Gateway による低コスト運用と、人が責任を持つ範囲の線引きを記録します。士業 / 中小企業の総務担当 · 2937字
  3. 月次KPIレポートを15分で読める形に。経営指標レポート生成エージェント売上・契約数・受注率などのKPI数値表を貼り付けるだけで、サマリ・注目すべき変化・来月の打ち手まで含んだ月次レポートの下書きを生成するAI エージェントの設計と運用を、Phasera が中小企業向けに解説します。中小企業の経営者 / 社内CFO · 3095字
  4. 案件と求職者の重なりを、説明可能なスコアで見える化するエージェント1 案件あたり 10 名の候補比較に 30 分以上かかる現場で、適合度を 0-100 のスコアと根拠つきの理由で出す AI エージェント。実装の核と動作確認を、Phasera が解説します。人材派遣・紹介業のキャリアアドバイザー · 2882字
  5. スカウト DM の一通目を 3 パターンに分解する、伴走型エージェント人材派遣・紹介の現場で 1 件 15-25 分かかるスカウト DM の下書きを、AI エージェントで「短い・標準・詳しい」の 3 パターンに分解。実装の核となる System Prompt と実出力を、Phasera が解説します。人材派遣・紹介業の代表 / リクルーター · 2921字
